# Instruções para o Claude - LEIA PRIMEIRO

## Ao iniciar uma nova sessão:

1. **LEIA** este arquivo e os outros na pasta `.claude/`
2. **LEIA** o `DEVLOG.md` - ele contém o histórico completo de tudo que foi feito
3. **LEIA** o `PROJECT_CONTEXT.md` - ele contém a visão geral do projeto
4. **CONTINUE** de onde parou segundo o DEVLOG

## Regras de trabalho:

1. **SEMPRE** adicione uma entrada no DEVLOG.md ao INÍCIO de cada sessão
2. **SEMPRE** atualize o DEVLOG.md ao FINAL de cada tarefa completada
3. **SEMPRE** registre arquivos criados/modificados
4. **SEMPRE** registre o próximo passo planejado
5. **USE** formato de data: [AAAA-MM-DD HH:MM]
6. **MARQUE** tarefas incompletas claramente com ⚠️

## Formato da entrada do DEVLOG:

```markdown
## [DATA] Título da Tarefa
**Status:** Em andamento / Concluído / ⚠️ Incompleto
**O que foi feito:**
- Item 1
- Item 2

**Problemas encontrados:**
- (se houver)

**Próximos passos:**
- Próxima tarefa a fazer

**Arquivos modificados:**
- caminho/do/arquivo (criado/modificado/deletado)
```
