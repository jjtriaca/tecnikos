# Regra: Painel Meta/Facebook sempre em portugues

## Contexto
O usuario (Juliano) acessa o painel da Meta (business.facebook.com e developers.facebook.com) com interface em **portugues brasileiro**. Quando orientar sobre navegacao no painel Meta, SEMPRE usar os nomes dos menus e campos em portugues.

## Regras
1. NUNCA usar nomes em ingles para menus, botoes e campos (ex: "System Users" -> usar "Usuarios do sistema")
2. NUNCA dizer que algo e um link clicavel se nao for (ex: numeros de ID azuis no business.facebook.com nao sao links)
3. Quando orientar navegacao, ser ESPECIFICO sobre onde clicar (menu lateral, aba, botao)
4. Diferenciar claramente entre business.facebook.com (Configuracoes do Negocio) e developers.facebook.com (Painel do Desenvolvedor)

## Onde encontrar Phone Number ID
- NAO esta em business.facebook.com
- Esta em developers.facebook.com -> App Tecnikos -> WhatsApp -> Configuracao da API
- Aparece abaixo do dropdown de selecao do numero
- E um numero longo tipo 142150505285xxxx (diferente do numero de telefone e do WABA ID)
